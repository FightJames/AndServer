注解

注解

AndServer 基于 Apt 技术，在项目编译时根据开发者的注解为开发者生成原生 Java 代码。因此开发者不需要担心注解的性能问题，只需要按照文档将一些注解加到一些类和方法上，然后启动服务器即可，不需要其它任何注册或者配置就可以看到效果。

理解和掌握这些注解非常简单，如果开发者没有使用过编译时注解的项目，也许 AndServer 是一个好的开始，不妨先看看入门。
